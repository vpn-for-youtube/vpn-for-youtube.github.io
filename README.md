# vpn-for-youtube.github.io

Статический SEO-лендинг с рейтингом VPN для YouTube: первое место — [BAZAVPN](https://baza-vpn.pro/), сравнение с известными сервисами (Proton VPN, Windscribe, Atlas VPN, Hide.me и др.).

## Публикация на GitHub Pages

1. Создайте репозиторий с именем **`vpn-for-youtube.github.io`** под аккаунтом `vpn-for-youtube`.
2. Загрузите содержимое этой папки в ветку **`main`** (корень репозитория — `index.html`, `styles.css`, `app.js` и остальные файлы).
3. В репозитории: **Settings → Pages → Build and deployment → Source**: ветка **main**, папка **/ (root)**.
4. Сайт будет доступен по адресу **https://vpn-for-youtube.github.io/**

## Поисковые системы

- Добавьте сайт в [Яндекс.Вебмастер](https://webmaster.yandex.ru/) и [Google Search Console](https://search.google.com/search-console).
- Укажите URL sitemap: `https://vpn-for-youtube.github.io/sitemap.xml`.
- Файлы верификации (например `google*.html`, `yandex_*.html`) положите в корень репозитория — GitHub Pages отдаст их как статику.

## Локальная проверка

Откройте `index.html` в браузере или поднимите любой статический сервер в этой директории.
